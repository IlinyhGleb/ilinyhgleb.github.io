В этом задании мы получаем цифровые данные из растрового изображения сканированной страницы учебника. 
После получения цифровых данных с ними можно работать в электронных таблицах (например MS Excel, Google Docs), в скриптах на различных языках программирования (Python, Java и т.д.). 
Можно эти данные аппроксимировать, находить зависимости, анализировать.

## Задание 1

1. Скачайте ({{< download file="/files/digitizer/excel_plot_digitizer_1.png" label="растровое изображение" >}}).

2. Откройте изображение в приложении **Paint**, или **Paint3d** или любом другом редакторе растровых изображений.

3. Обрежьте изображение. На изображении должен остаться график, оси, подписи осей.

## Задание 2

1. Откройте в браузере один из сайтов - сервисов оцифровки изображений
   [https://apps.automeris.io/wpd4/](https://apps.automeris.io/wpd4/)
   [https://plotdigitizer.com/app](https://plotdigitizer.com/app)

2. Загрузите на сайт полученный в **Задании 1** график.

3. Выберите опорные точки (координаты X1, X2, и координаты Y1, Y2 на известных Вам местах - пересечениях сетки)

   ***Примечание:*** *Мы знаем значения в на пересечениях сетки, но не знаем значения на графике, поэтому мы выбираем опорные точки*
   
4. Проставьте точки по равномерно на кривой от начала (слева) до конца (справа)
5. Экспортируйте данные в формате .csv

	***Примечание: *** *.csv - формат файлов для хранения табличных данных. 
	Формат удобен для передачи импорта/экспорта данных, передачи данных между приложениями. 
	Файл .csv можно открыть с помощью любого текстового редактора. 
	Данные записаны в столбцы. 
	Данные из разных столбцов разделены с помощью разделителя, обычно разделитель - это запятая (,) или точка с запятой (;). 
	В начале файла присутствует строка заголовков столбцов. 
	Имена столбцов также разделены с помощью разделителя.*
