## Расчёт балки

### Геометрия

Создадим геометрию цилиндрической сваи и окружающего грунта в приложении *Design Modeler*.

{{< media/video src="/videos/ansys/wb4/open_dm.mp4" width="800" >}}

Создадим макет (Sketch) сваи

{{< media/video src="/videos/ansys/wb4/sketch_pile.mp4" width="800" >}}

Создадим макет (Sketch) окружающего грунта

{{< media/video src="/videos/ansys/wb4/sketch_soil.mp4" width="800" >}}

На основе макета сваи создадим её тело вращением вокруг оси с помощью инструмента *Revolve*

{{< media/video src="/videos/ansys/wb4/revolve_pile.mp4" width="800" >}}

Аналогичным образом создадим тело грунта

{{< media/video src="/videos/ansys/wb4/revolve_soil.mp4" width="800" >}}

Вычтем из тела грунта тело сваи с помощью инструмента *Boolean* (опция *Subtract*). Для того, чтобы тело сваи не исчезло, используем опцию *Preserve Tool Bodies?* - *Yes* (Сохранить вычитаемые тела)

{{< media/video src="/videos/ansys/wb4/subtract_pile_soil.mp4" width="800" >}}

Полученная геометрия уже может быть использована для расчёта, однако она имеет сложную топологию.
Для того, чтобы модель было легче разбить на конечные элементы, а так же для получения более качественной конечно-элементной сетки разобьём модель на более простые тела.

Чтобы разрезать тело плоскостью, создадим дополнительную плоскость на конце сваи.

{{< media/video src="/videos/ansys/wb4/add_plane.mp4" width="800" >}}

Для горизонтального разрезания тела грунта используем инструмент *Slice* с опцией *Slice by Plane*

{{< media/video src="/videos/ansys/wb4/slice_by_plane.mp4" width="800" >}}

Для вертикального разрезания тела грунта по цилиндрической поверхности сваи используем инструмент *Slice* с опцией *Slice by Surface*

{{< media/video src="/videos/ansys/wb4/slice_by_surface.mp4" width="800" >}}

Объедим все четыре тела в компоненту с именем *грунт-свая*

{{< media/video src="/videos/ansys/wb4/part.mp4" width="800" >}}

Настроим создание сетки для верхних двух тел с указанием источника сетки в виде поверхности, от которой пойдёт вытягивание. 
Двухмерная сетка на поверхности будет создана в первую очередь.

## Задачи для самостоятельного решения

1. С помощью запуска задачи с разными размерами грунта определите минимально допустимые размеры модели. 
Минимально допустимыми размерами являются такие, при которых граничные условия не влияют на решение, и наоборот, деформирование в центре не влечёт деформирование на границе, т.е. на границе нет напряжений. 
2. Измените геометрию сваи заменив конец сваи с цилиндрического на конусообразный. Сравните результаты двух моделей.
